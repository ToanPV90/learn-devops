# chage

<br>

## NAME

chage - change user password expiry information

<br>

## SYNOPSIS

> chage [options] user

<br>

## OPTIONS

* -l, --list
  * Show account aging information.
  
<br>

## EXAMPLE

* To get the password expiry information of the current user using the system, you can use

```bash
chage -l $USER
.
```
